# Klarheit Testimonial 05

Mixed-media testimonial gallery with a large section heading, supporting call-to-action, portrait cards, and a featured quote card in a horizontal grid. Part of the Klarheit collection for modern agencies and creative studios.

## Features

- **Horizontal grid layout** with responsive auto-fit columns
- **Featured testimonial card** that spans 2 columns with accent color background
- **Portrait images** with circular cropping for personal touch
- **Inline editable text** for eyebrow, heading, and CTA
- **Style controls** for typography, colors, spacing, and button variant
- **CMS integration** for managing testimonials with name, role, image, quote, and featured flag
- **Hover effects** with subtle lift and shadow
- **Responsive design** that adapts to mobile screens

## Panel checklist

- Keep visible section-level text inline editable with `EditableText`.
- Put repeatable testimonial content in the top-level `cms` block.
- Keep `Update Styles` available with Layout controls (section padding, card gap).
- Use style guide token controls for typography, color, and buttons.
- Add `panel: "animation"` only when the section has real animation controls.
