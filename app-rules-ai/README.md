# App Rules AI

These are the canonical rules for AI agents and language models working in this repo.

Read the files that match the task before changing code:

- `whole-app/` — app-wide roadmap, deployment, database, and process rules.
- `making-a-section/` — rules for creating or updating library sections and their builder panels.

The old `.windsurf/rules` folder has been replaced by this folder. If a tool needs a different rules location, mirror these files there, but keep `app-rules-ai` as the source of truth.
